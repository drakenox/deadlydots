## Programs

  - OS : NixOs
  - WM : Openbox
  - Editor : Nvcodark
  - Terminal : Alacritty
  - Shell : Hilbish
  - Launcher : Rofi

## Why NixOS?

- Declarative package and configuration management
- Lots of packages
- Rollbacks of configurations (When you mess something up, like the kernel)
- Reliable system
- Reproducible environment
